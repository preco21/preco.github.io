# preco21.github.io (prevl.io)

> :sparkles: Prevl IO  :sparkles:

This branch contains source of [prevl.io](https://prevl.io) website.

## Installation

```bash
$ git clone https://github.com/preco21/preco21.github.io.git
$ cd preco21.github.io
$ npm install
```

## Usage

```bash
# Serve website
$ npm start

# Build website
$ npm run build
```
